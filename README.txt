== Description ==

<h3>Top List</h3>This plugin is right for you if you want to create listing like in Listverse or Toptenz. You can create List easily. You can add list items as you want. Default template layout is set by the plugin but those templates could be overridden from the theme. Several hooks are available for you to extend the plugin feature.



== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'top-list.zip' from your computer
1. Upload
1. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =

1. Extract 'top-list.zip' to your computer
1. Upload the 'top-list' directory to your 'wp-content/plugins' directory
1. Activate the plugin on the WordPress Plugins dashboard
